\textbf{Definition 1.}
A planar integral point set (PIPS) is a set $M$ of noncollinear points in plane $\mathbb{R}^2$
such that for any pair of points $M_1, M_2\in M$ the distance between $M_1$ and $M_2$ is integral.
Every PIPS is finite [1].

Some general constructions of PIPS are known,
e.g. a concyclic PIPS of an arbitrary power [2]
and a PIPS of an arbitrary power $n$ such that $(n-1)$ points are collinear [3].
However, there is no known algorithm to construct a PIPS of arbitrary power $n$ which is contained in 2 parallel straight lines,
no more than $n-2$ points on each line.

Examples of such PIPS for $n=7,8,9$ can be found in [4,5].
Below an example for $n=10$ is given.
We use notation
\begin{equation*}
	\sqrt{p}/q * \{ (x_1,y_1), ..., (x_n, y_n)  \}
	:=
	\left\{ \left(\frac{x_1}{q},\frac{y_1\sqrt{p}}{q}\right), ..., \left(\frac{x_n}{q},\frac{y_n\sqrt{p}}{q}\right)  \right\}
	.
\end{equation*}
